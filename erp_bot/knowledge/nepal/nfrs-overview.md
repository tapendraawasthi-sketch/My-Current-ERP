# NFRS Overview (Nepal Financial Reporting Standards)

Nepal adopted **NFRS** (Nepal Financial Reporting Standards) aligned with IFRS for entities above prescribed thresholds.

## Applicability (general)

| Entity type | Typical framework |
|-------------|-------------------|
| Listed / large entities | Full NFRS |
| SMEs (below threshold) | NFRS for SMEs |
| Micro entities | Nepal Accounting Standards (NAS) simplified |

## Key principles

- **Accrual basis** — recognize income/expense when earned/incurred.
- **Going concern** — financial statements assume continuity.
- **Materiality** — disclose items that could influence decisions.
- **Consistency** — same accounting policies period to period.

## Common standards referenced

- NFRS 15 — Revenue from contracts with customers
- NFRS 16 — Leases
- NAS 2 — Inventories (for smaller entities)
- NAS 16 — Property, plant and equipment

For conceptual framework questions (assets, liabilities, equity), IFRS Conceptual Framework applies as interpretive guidance.
