# UI-7 — Document Canvas Spec

Premium financial-document canvas (not a generic admin form).

## Regions

1. **Context** — title, company, FY, draft/post status, sync chip  
2. **Header fields** — party/accounts, date, reference, warehouse, payment mode  
3. **Entry** — line grid / allocation / transfer / voucher grid  
4. **Financial summary** — only fields supported by document type  
5. **Supporting** — narration, notes  
6. **Inspector** — preview/warnings/evidence (collapsible)  
7. **Actions** — draft / post / cancel — truthful labels  

No giant floating card; continuous entry flow; material tax/settlement effects remain visible.
