# Accessibility Foundations

- `.ds-focus-ring` uses `:focus-visible`
- `VisuallyHidden` / `.sr-only`
- Min hit targets via `--ds-hit-target-min`
- IconButton requires `aria-label` in TypeScript
- FormField wires `aria-describedby` + `aria-invalid`
- `prefers-reduced-motion` respected in foundations and Spinner/Skeleton
