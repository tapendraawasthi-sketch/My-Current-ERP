# Icon Policy

- General family: Lucide (existing) — do not mass-replace in UI-1
- Wrapper sizes: 16 / 20 / 24; stroke via `currentColor`
- Icon-only controls require accessible names
- Decorative: `aria-hidden`
- No sparkle-as-AI, no emoji, no religious/national emblems

## Custom icons (minimal set)

`OrbixIcon`, `NprIcon`, `DualDateIcon`, `LedgerIcon`, `ReconciliationIcon`, `SyncConflictIcon` in `src/design-system/icons/`
