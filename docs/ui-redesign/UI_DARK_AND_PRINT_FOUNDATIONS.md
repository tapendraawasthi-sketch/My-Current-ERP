# Dark & Print Foundations

## Dark mode

`data-theme` / `data-ds-theme` = `dark`. Semantic tokens redefined — not mechanical invert.

**Claim:** NEW DESIGN-SYSTEM FOUNDATIONS AND PRIMITIVES SUPPORT DARK MODE  
(Not full-application dark-mode completion.)

## Print

`.ds-print-root`, `.ds-no-print`, `.ds-print-only`, page-break utilities, repeated thead helper. Light background, no shadows, no dark print.
